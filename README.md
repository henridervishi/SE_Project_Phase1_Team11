# Project Details

## Project Title: Student Roommate Finder

## Problem Statement:
Many students who move to Tirana for university struggle to find 
suitable and trustworthy roommates. The search process is currently 
scattered across social media groups and word of mouth, making it 
inefficient, unsafe, and stressful for students looking for affordable 
shared housing.

## Proposed Solution:
A web platform where students can create profiles, list available rooms 
or search for roommates based on preferences such as location, budget, 
lifestyle habits, and university. The platform enables direct 
communication between students and centralizes the entire roommate 
search process in one safe and organized place.

## Project Scope:
- **Aim:** To simplify and digitize the process of finding compatible 
  roommates for students in Tirana.
- **Objectives:**
  1. Allow students to register and create detailed personal profiles.
  2. Enable users to post room listings with photos, price, and location.
  3. Provide search and filter functionality by budget, location, and lifestyle.
  4. Allow direct messaging between interested students.
  5. Include a verification step to ensure users are genuine students.

## Application Description:
The application targets university students in Albania who are looking 
for shared housing or roommates. Users can sign up, build a profile 
describing their habits and preferences, and either post a room listing 
or browse existing ones. Key features include profile creation, room 
listings with photos, advanced search filters, and an in-app messaging 
system. The goal is to make finding a roommate safer, faster, and 
more organized for students across Tirana.
